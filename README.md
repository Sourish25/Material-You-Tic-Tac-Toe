# TikTac

A modern Android Tic-Tac-Toe game.

## Features
- Classic Tic-Tac-Toe gameplay.
- Single player vs AI (implied, adjust if multiplayer).
- Clean and responsive UI.

## Getting Started

### Prerequisites
- Android Studio
- Android SDK

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/tiktac.git
   ```
2. Open the project in Android Studio.
3. Sync Gradle and build the project.
